수업 때 다루진 않지만 더 알아두면 좋은 개념

1. 함수와 배열(배열을 매개변수로 쓸 경우, 반환형으로 쓸 경우)
=> 배열은 포인터이므로 배열을 매개변수로 받고 해당 배열 안의 값을 바꾸면 main에서도 바뀜
=> 매개변수로 배열 넘길 때 배열의 크기도 같이 넘겨야 함. 

2. 구조체와 배열
=> 구조체를 이용해서 배열 만들 수 있음. 즉 학생배열, 직원배열 등이 가능

3. 구조체와 함수
=> 구조체를 매개변수로 하거나 반환형으로 할 수 있음

4. 구조체와 포인터
=> int 포인터 뿐 아니라 구조체 포인터도 존재함.
사용 양식이 조금 독특함.
int 포인터 : int* p = &a; //*p 하면 변수 a의 값 보여줌
구조체 포인터 Student* s = &stu; // s->name 등으로 접근(화살표)

5. 구조체 동적 할당
=> 구조체와 포인터가 연관이 있으므로 마찬가지로 동적 할당도 가능
